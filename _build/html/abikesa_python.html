import matplotlib.pyplot as plt
import numpy as np

# Create data for the skill and challenge levels
skill_levels = np.linspace(0, 10, 100)
challenge_levels = np.linspace(0, 10, 100)

# Define the flow channel boundaries
flow_channel = skill_levels

# Adjust the phase and amplitude of the sinusoid wave
phase = np.pi / 16
amplitude = 1.5
sinusoid = flow_channel + np.sin(skill_levels + phase) * amplitude

# Define the yellow zone boundaries, making it wider
yellow_zone_low = skill_levels - 1.5  # Adjust this value to make the yellow zone wider or narrower
yellow_zone_high = skill_levels + 1.5 # Adjust this value to make the yellow zone wider or narrower

# Plotting
plt.figure(figsize=(15, 10))

# Plot the anxiety and boredom areas
plt.fill_between(skill_levels, yellow_zone_high, 10, color='orangered', alpha=0.3, label='Anxiety Zone', interpolate=True)
plt.fill_between(skill_levels, 0, yellow_zone_low, color='lime', alpha=0.3, label='Boredom Zone', interpolate=True)
plt.fill_between(skill_levels, yellow_zone_low, yellow_zone_high, color='yellow', alpha=0.3, label='Flow Zone', interpolate=True)

# Plot the sinusoid function with the diagonal as its axis
plt.plot(skill_levels, sinusoid, color='purple', linestyle='-')

# Add arrowhead to the sinusoid line
plt.arrow(skill_levels[-2], sinusoid[-2], skill_levels[-1] - skill_levels[-2], sinusoid[-1] - sinusoid[-2],
          color='purple', length_includes_head=True, head_width=0.15, head_length=0.3)

# Add an "X" marker at the specified coordinate
plt.scatter(1.5, 3.5, color='red', s=300, marker='X')

# Set plot labels and title
plt.xlabel('Skill-level', fontsize=18)
plt.ylabel('Challenge-level', rotation='vertical', fontsize=18)

# Set plot limits and grid
plt.xlim(0, 10)
plt.ylim(0, 10)
plt.grid(True)

# Set tick labels
tick_labels = ['0', '2', '4', '6', '8', '10']
plt.xticks(np.linspace(0, 10, 6), tick_labels)
plt.yticks(np.linspace(0, 10, 6), tick_labels)

# Add text annotations to label the areas without shaded background

plt.text(8, 3, 'Prohibitum/Goliath', color='black', ha='center', va='center', fontsize=18)

# Save the figure
plt.savefig('maluminse.png', dpi=300)  # Saves the figure as a high-resolution png file

# Display the plot
plt.show()

Hi, I would appreciate any advice! I would like to create a figure that shows the percentage of heart transplantations performed in each UNOS region. What is the easiest command to achieve this? It should be a heatmap and I want to change the color of 11 UNOS regions according to stratified percentages. The attached figure shows 11 UNOS regions.

Creating a heatmap to show the percentage of heart transplantations performed in each UNOS region is a great way to visualize geographic disparities or trends in medical procedures. Here’s how you can achieve this in Python using the popular library matplotlib. This library is very capable and versatile for creating visualizations like heatmaps.

Here’s a step-by-step guide to creating your heatmap:

Step 1: Prepare Your Data#

You’ll need to have your data organized, typically in a CSV or Excel format, where you have two columns: one for the UNOS region and another for the percentage of heart transplantations. For this example, let’s assume the data is in a CSV file called transplant_data.csv.

Step 2: Install Required Libraries#

Make sure you have the necessary Python libraries installed. You can install them using pip if you don’t have them already:

pip install pandas matplotlib

Step 3: Load Your Data#

Use pandas to load your data and matplotlib to create the heatmap:

import pandas as pd
import matplotlib.pyplot as plt

# Load data
data = pd.read_csv('transplant_data.csv')

# Assuming the data file has columns 'Region' and 'Percentage'
data = data.pivot("Region", "Percentage")

Step 4: Create the Heatmap#

To create a heatmap where colors correspond to the percentages of heart transplantations:

# Create a heatmap
plt.figure(figsize=(10, 8))
heatmap = plt.pcolor(data, cmap='Reds')
plt.colorbar(heatmap)

# Setting labels
plt.xticks(range(len(data.columns)), data.columns)
plt.yticks(range(len(data.index)), data.index)
plt.title('Heatmap of Heart Transplantations by UNOS Region')

# Show the plot
plt.show()

In this script, the cmap='Reds' parameter in plt.pcolor() determines the color scheme, which can be adjusted to different colormaps such as ‘Blues’, ‘Greens’, etc., based on your preference. This will stratify the colors by the percentages, providing a clear visual distinction between regions with different transplantation rates.

If you have any questions about the code or need further customization, feel free to ask!
